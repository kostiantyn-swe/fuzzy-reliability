# Operator Reliability Assessment under Data Uncertainty

A web application for predicting human-operator reliability indicators in 
human-machine systems, based on the functional-structural theory of ergatic 
systems (Gubinsky school) combined with fuzzy set theory.

🔗 **Live demo:** https://fuzzyreliability.pythonanywhere.com/

## Features
- Calculation of 4 reliability indicators for 5 typical functional structures
- 13 membership function types, 65 ready-to-use model combinations
- REST API (/api/evaluate) with input validation
- Results verified against MATLAB Fuzzy Logic Toolbox

## Tech Stack
Python · Flask · REST API · NumPy · pytest

## Testing
84 unit tests, 90% code coverage

## Context
Master's thesis project, Sumy State University, 2026. 
Results have been implemented in a real business workflow and presented at 
the International Scientific Conference IMA-2026 (Sumy–Astana).

# Оцінювання надійності оператора в умовах нечіткості даних

Веб-застосунок для прогнозування показників надійності людино-машинних 
систем на основі функціонально-структурної теорії ерготехнічних систем 
(школа Губінського) та апарату нечітких множин.

🔗 **Демо:** https://fuzzyreliability.pythonanywhere.com/

## Можливості
- Розрахунок 4 показників надійності для 5 типових функціональних структур
- 13 типів функцій належності, 65 готових комбінацій моделей
- REST API (/api/evaluate) з валідацією вхідних даних
- Верифікація результатів проти MATLAB Fuzzy Logic Toolbox

## Технології
Python · Flask · REST API · NumPy  · pytest

## Тестування
84 unit-тести, 90% покриття коду

## Контекст
Кваліфікаційна робота магістра, СумДУ, 2026. 
Результати впроваджено в реальний бізнес-процес та представлено на 
Міжнародній науковій конференції ІМА-2026 (Суми–Астана).
